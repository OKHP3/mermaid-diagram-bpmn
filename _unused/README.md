# _unused — Replit template scaffolding

This directory contains artifacts and libraries that were generated by the Replit monorepo template but are not used by the `bpmn-beta` prototype. They are kept here rather than deleted in case they become useful later.

## Contents

- `lib/api-spec/` — OpenAPI spec with a single health-check route; generated by template codegen
- `lib/api-client-react/` — React Query hooks auto-generated from the OpenAPI spec
- `lib/api-zod/` — Zod schemas auto-generated from the OpenAPI spec
- `lib/db/` — PostgreSQL + Drizzle ORM schema; provisioned but not used by the BPMN frontend

Note: The `artifacts/api-server` Express 5 artifact is kept in its original location
(`artifacts/api-server/`) rather than moved here, because Replit's environment has it
registered as a workflow artifact. Moving it would break the Replit dev environment.

## Safe to delete?

Yes, everything in this directory is safe to delete permanently if you are certain you
will not need a backend API or database for the BPMN prototype. The frontend is
fully client-side and has no dependency on any of these packages.
